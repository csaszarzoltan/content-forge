# ContentForge

AI-powered content platform with brand voice customization.

Parse, manage, and inject brand voice profiles into LLM prompts for consistent brand-aligned content generation.

## Features

- 🎨 **Brand Voice Customization** — Define and manage brand voice profiles with vocabulary rules, scenario tones, and formatting preferences
- 🤖 **LLM Prompt Binding** — Inject voice profiles into content prompts with system prompt generation
- 📊 **Compliance Scoring** — Score generated content against brand guidelines
- 🔄 **Multi-Brand Management** — CRUD operations across multiple brand profiles
- 📝 **Voice Scoping** — Per-user and per-project voice resolution
